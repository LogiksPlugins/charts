# Charts

This plugin helps load various chart vendors and required functionality, including data transformation code for data from Logiks Service Data Format to the required library format.

### Supported libraries
- [X] ChartJS
- [X] JQuery Sparkline
- [X] JQPlot
- [X] Flot
- [X] Google Charts
- [X] Highchart
- [X] ApexCharts
- [X] eCharts (Apache)
- [ ] C3/D3
